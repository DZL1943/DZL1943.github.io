## 命题

- 直言命题
    - 单称肯定命题
    - 单称否定命题
    - 全称肯定命题(SAP)
    - 全称否定命题(SEP)
    - 特称肯定命题(SIP)
    - 特称否定命题(SOP)
- 复言命题
    - 联言命题(且) `非(P 且 Q) = (非 P) 或 (非 Q)`
    - 选言命题(或)
        - 相容 `非(P 或 Q) = (非 P) 且 (非 Q)`
        - 不相容(要么..要么..)
    - 假言/条件命题 充分 => 必要; 肯前必肯后, 否后必否前(逆否等价).
        - 充分(如果..那么.., 只要..就.., 凡是..都..) 前推后
        - 必要(只有..才.., 除非..否则不..) 后推前

## 关系

- 矛盾关系:全称肯定(全肯)与特称否定(特否)、全称否定(全否)与特称肯定(特肯)命题之间的关系.这两对命题总是具有相反的真值,即如果一个为真,则另一个必定为假,反之亦然.
- 上反对关系:两个全称命题(一个是肯定的,另一个是否定的)之间的关系.特点是这两个命题中至少有一个是假的,但它们也可以同时为假.
- 下反对关系:两个特称命题(一个肯定,另一个否定)之间的关系.特点是这两个命题中至少有一个是真的,但它们也可能同时为真.
- 差等关系:全称命题和相应的特称命题之间(无论是肯定还是否定形式)的关系.特点是两者既可以同时为真,也可以同时为假.这种关系是单向推导的,即从全称命题可以推导出特称命题,但反向则不成立.

```
A (全称肯定) —————— 反对 —————— E (全称否定)
  |                         |
差等                      差等
  |                         |
I (特称肯定) ————— 下反对 ————— O (特称否定)
```
