## 各部位训练方式

- [x] 肩: 推举、平举、飞鸟、俯身飞鸟
- [x] 臂: 弯举、锤式弯举、俯身臂屈伸/过头臂屈伸
- [x] 胸: 卧推、飞鸟、窄推
- [ ] 腹: 卷腹、转体、反抬腿
- [x] 背: 正反手划船、抱拉、俯身反向飞鸟
- [ ] 臀
- [x] 腿: 深蹲、硬拉、箭步蹲、提踵

## 训练计划

### 三分化

- 推: 胸、肩、三头
- 拉: 背、二头
- 腿

一周循环 2 次, 间隔一天

### (四分化)训练计划 1

1. 胸 + 三头(臂屈伸)
2. 背 + 二头(弯举)
3. 休息
4. 肩
5. 臀腿
6. 休息
7. 有氧或腹

### 训练计划 2

1. 胸 + 背
2. 休息
3. 肩 + 手
4. 休息
5. 腿 + 腹
6. 休息
7. 休息

### (五分化)训练计划 3

1. 胸
2. 背
3. 肩
4. 休息
5. 腿
6. 手臂
7. 休息
