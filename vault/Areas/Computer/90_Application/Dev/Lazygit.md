# Lazygit

一个终端下的 git 客户端.  
支持鼠标、快捷键操作.

主要分为以下 5 大块. 每个模块下有特定的快捷操作

## Status

- Edit config file
- Check for update
- Switch to a recent repo

## Files、Worktrees、Submodules

- Stage
- Commit
- Edit
- Stash
- Discard
- Reset

## Local branches、Remotes、Tags

- Checkout
- New branch
- Delete
- Rebase
- Reset
- Upstream

## Commits、Reflog

- Squash
- Fixup
- Reword
- Drop
- Edit
- Amend
- Checkout
- Reset

## Stash

- Apply
- Pop
- Drop
