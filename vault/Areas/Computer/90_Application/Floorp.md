Floorp 是一个基于 Firefox ESR 的来自日本开发者的第三方发行版, 内置 TST 和 Gesturefy 插件.

外观层面的关键设置:
- Lepton UI (只有选这个才能进一步设置)
    - Lepton 设置
    - 自动隐藏浏览器元素
        - ~~自动隐藏标签栏~~: 严格来说只是隐藏了标签栏的文字
        - 自动隐藏工具链
        - **自动隐藏侧边栏**
    - 管理浏览器元素
        - 隐藏标签栏: 效果不太好
    - 位置调整
    - 地址栏
    - 标签栏
    - 侧边栏
- 标签栏样式
    - 横向标签栏: 选这个就够了
    - 多行标签栏
    - 垂直标签栏: 类似 Edge, 非 TST, 不支持树形
- 标签栏
    - **优化浏览器以适应垂直标签栏**
