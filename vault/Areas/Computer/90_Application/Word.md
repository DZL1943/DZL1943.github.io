## Tips

### 分节及其他特殊格式

| 代码    | 含义            | 用途                  |
| :---- | :------------ | :------------------ |
| `^p`  | 段落标记（硬回车）     | 分段、删空行（`^p^p`→`^p`） |
| `^l`  | 手动换行符（软回车）    | 网页复制来的假换行，需换成 `^p`  |
| `^t`  | 制表符           | 替换为空格，或统一制表位        |
| `^m`  | 手动分页符         | 删除强制分页              |
| `^b`  | 分节符           | 删除/替换分节符（慎用，会丢格式）   |
| `^w`  | 空白区域（空格/制表混合） | 一键清理网页复制的混乱空白       |
| `^g`  | 图形/图片         | 批量选中图片改格式           |
| `^?`  | 任意单个字符        | 通配符匹配               |
| `*`   | 任意字符串         | 配合通配符使用             |
| `^&`  | 查找内容本身        | 替换时保留原内容，只改格式       |
| `^c`  | 剪贴板内容         | 替换为复制的内容            |
| `^9`  | 制表符（兼容旧版）     | 同 `^t`              |
| `^13` | 段落标记（ASCII 码）  | 同 `^p`，VBA 常用        |
| `^10` | 换行符（Unix 风格）   | 从 Linux/Mac 复制的文本     |
