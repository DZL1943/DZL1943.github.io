## Word

### 分节及其他特殊格式

| 代码    | 含义            | 用途                  |
| :---- | :------------ | :------------------ |
| `^p`  | 段落标记（硬回车）     | 分段、删空行（`^p^p`→`^p`） |
| `^l`  | 手动换行符（软回车）    | 网页复制来的假换行，需换成 `^p`  |
| `^t`  | 制表符           | 替换为空格，或统一制表位        |
| `^m`  | 手动分页符         | 删除强制分页              |
| `^b`  | 分节符           | 删除/替换分节符（慎用，会丢格式）   |
| `^w`  | 空白区域（空格/制表混合） | 一键清理网页复制的混乱空白       |
| `^g`  | 图形/图片         | 批量选中图片改格式           |
| `^?`  | 任意单个字符        | 通配符匹配               |
| `*`   | 任意字符串         | 配合通配符使用             |
| `^&`  | 查找内容本身        | 替换时保留原内容，只改格式       |
| `^c`  | 剪贴板内容         | 替换为复制的内容            |
| `^9`  | 制表符（兼容旧版）     | 同 `^t`              |
| `^13` | 段落标记（ASCII 码）  | 同 `^p`，VBA 常用        |
| `^10` | 换行符（Unix 风格）   | 从 Linux/Mac 复制的文本     |

### 引用其他 sheet 中的行

1. 选择目标单元格
2. 输入=
3. 切换到对应 sheet, 用鼠标选择区域, 回车
4. 处理空值例如 `IF(Sheet2!A5:P7="", "", Sheet2!A5:P7)`

例子: A5:P7、A5:P9、A5:P7, 从第六行开始.
```
IFERROR(
  IF((ROW()-5)>SUMPRODUCT(ROWS(Data1), ROWS(Data2), ROWS(Data3)), "",
  IF((ROW()-5)<=ROWS(Data1),
    INDEX(Data1, ROW()-5, COLUMN()) & "",
  IF((ROW()-5)<=ROWS(Data1)+ROWS(Data2),
    INDEX(Data2, (ROW()-5)-ROWS(Data1), COLUMN()) & "",
    INDEX(Data3, (ROW()-5)-ROWS(Data1)-ROWS(Data2), COLUMN()) & ""
  ))), 
"")
```
