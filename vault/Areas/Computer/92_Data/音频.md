## 格式

| 格式名称           | 压缩类型 | 最高采样率/位深支持             | 典型比特率范围                  | 主要用途/特点                                               |
| :------------- | :--- | :--------------------- | :----------------------- | :---------------------------------------------------- |
| **WAV**        | 无    | 理论上极高 (如 768kHz/32bit) | 1411 kbps (CD) 或更高       | 专业音频编辑、CD 存储、Windows 系统通用无损容器                         |
| **AIFF**       | 无    | 理论上极高                  | 1411 kbps (CD) 或更高       | 专业音频编辑、CD 存储、macOS 系统通用无损容器                           |
| **FLAC**       | 无损压缩 | 非常高 (如 384kHz/32bit)   | ~600 - 1100 kbps (CD音质下) | **主流无损格式**，高保真音乐存储/播放、音乐收藏、节省空间的无损                    |
| **ALAC**       | 无损压缩 | 非常高                    | 类似 FLAC                  | Apple 生态系统无损 (iTunes, Apple Music无损层)                 |
| **APE**        | 无损压缩 | 高                      | 压缩率有时略高于 FLAC            | 压缩率高，但编解码慢、兼容性不如 FLAC                                 |
| **MP3**        | 有损压缩 | 通常 48kHz/16bit         | 32kbps - **320kbps**     | **最广泛兼容的有损格式**，音乐下载、流媒体(旧)、通用播放                       |
| **AAC**        | 有损压缩 | 96kHz / 多声道            | 8kbps - **256kbps+**     | **MP3 的现代替代者**，效率更高， iTunes、YouTube、Spotify免费层、手机录音主流 |
| **OGG Vorbis** | 有损压缩 | 高                      | ~64kbps - 500kbps        | 开源高效，Spotify 使用，游戏音频                                  |
| **Opus**       | 有损压缩 | 48kHz / 16bit (音乐)     | **6kbps - 510kbps**      | **超低延迟、极低比特率高效**，网络语音/视频通话、游戏语音、实时流媒体首选               |
| **WMA**        | 有损压缩 | 高                      | 可变                       | 微软旧格式，流行度下降                                           |
