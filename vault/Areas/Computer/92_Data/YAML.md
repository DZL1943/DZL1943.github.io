单行注释: `#`

标量 scalar
- boolean
- null
- number
- string
    - 单引号
    - 双引号
    - 多行
        - `|`
            - `|-`
            - `|+`
        - `>`
            - `>-`
            - `>+`
- datetime
- !!binary

序列 seq: 缩进形式 `- `, json 形式 `[,]`
- 标量序列 `[,]`
- 映射序列 `[{},]`
- 序列序列 `[[],]`

集合 set: 缩进形式 `? `, json 形式 `{,}`

映射 map: 缩进形式 `: `, json 形式 `{:,}`
- 标量 -> 标量 `{k:v}`
- 标量 -> 序列 `{k:[]}`
- 标量 -> 映射 `{k:{}}`
