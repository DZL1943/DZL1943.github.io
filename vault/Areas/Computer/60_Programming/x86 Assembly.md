## 寄存器

- 段寄存器
    - CS 代码段
    - DS 数据段
    - SS 栈
    - ES 附加段
    - FS
    - GS
- 通用寄存器 (GPR)
  
  | 寄存器 | 传统用法                     |
  | ------ | ---------------------------- |
  | EAX    | 累加器                       |
  | EBX    | 内存指针, 基址寄存器         |
  | ECX    | 循环控制, 计数器             |
  | EDX    | 整数乘除法                   |
  | ESI    | 串指令的源指针, 索引寄存器   |
  | EDI    | 串指令的目标指针, 索引寄存器 |
  | EBP    | 栈指针                       |
  | ESP    | 栈帧基址指针                 |
  
- EFLAGS

  | 13    |              | IOPL |      |
  | 14    |              | NT   |      |

- 标志操纵
