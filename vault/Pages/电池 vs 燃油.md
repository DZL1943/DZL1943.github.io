| 维度       | 电池                                        | 燃油                         |
|:---------- |:------------------------------------------- |:---------------------------- |
| 能量密度   | 更差(约 250-300 Wh/kg)                      | 更好(汽油约 12,000 Wh/kg)    |
| 使用寿命   | 更差(衰减、循环次数有限)                    | 更好(机械结构成熟耐用)       |
| 补能效率   | 更差(充电 30min+ vs 加油 3min)              | 更好                         |
| 功率/效率  | 更好(电机效率 90%+,瞬时扭矩大)              | 较差(内燃机效率 20-40%)      |
| 温度敏感性 | 更差(低温续航腰斩,高温易过热)               | 较好                         |
| 能量来源   | 依赖电网(清洁能源占比决定环保性)            | 依赖石油开采/炼化            |
| 基础设施   | 充电桩覆盖仍不足,长途焦虑                   | 加油站网络成熟 ubiquitous    |
| 维护成本   | 更好(少保养项目:无机油、火花塞、变速箱油等) | 更高(定期保养项目多)         |
| 购置成本   | 电池成本高,车价通常更贵                     | 相对便宜(同级别)             |
| 使用成本   | 更好(电费 vs 油费,1/3~1/5 成本)             | 更高(油价波动大)             |
| 静谧性     | 更好(无发动机噪音振动)                      | 较差                         |
| 安全性     | 热失控风险(撞击/穿刺易起火,难扑灭)          | 燃油泄漏火灾相对易控制       |
| 环保性     | 使用阶段零排放,全生命周期看电池制造污染大   | 使用阶段排放高,产业链成熟    |
| 残值/二手  | 更差(电池衰减担忧,技术迭代快)               | 相对稳定                     |
| 重量       | 更差(电池组沉重,影响操控/能耗)              | 较轻                         |
| 空间布局   | 灵活(电池平铺底盘,前后储物空间多)           | 受发动机/变速箱/油箱布局限制 |
