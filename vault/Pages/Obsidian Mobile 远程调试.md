# Obsidian Mobile 远程调试

1. 手机开启 USB 调试或无线调试
2. 用类似 scrcpy 的工具连接电脑
3. 打开 obsidian
4. 以 Edge 为例, 打开 `edge://inspect/#devices`, 找到对应的设备, 点击 inspect, 之后就是熟悉的 DevTools 界面了
