# 我的笔记软件选择史
%% [title2list|list2table] %%
## 阶段一: OneNote

- 竞品: Apple Notes, Agenda(iOS), Drafts, Bear, MWeb
- 优点: 免费, 全平台, 云存储&同步.
- 不足: 富文本, 不支持 markdown
- 时间: 2016?
## 阶段二: Notion

- 竞品: 飞书, Craft
- 优点: 
    - 免费
    - 全平台, 云存储&同步
    - Markdown, Block, Database
- 不足: 网络
- 时间: 2018?
## 阶段三: Obsidian

- 竞品: Emacs、VSCode、Typora, 思源、Logseq、Anytype
- 优点: 
    - 免费
    - 全平台
    - markdown 文件透明存储 (数据与软件分离)
    - 插件生态
- 不足: Block
- 时间: 2021?
