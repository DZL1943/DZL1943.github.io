## 安装

我的世界分为 Java 版和基岩版两个版本, 可以简单理解为电脑版和手机版.

### Java 版(JE)

- 1.21(下个版本)
- 1.20.1(当前推荐版本)
- 1.19
- 1.17
- 1.12

Java 版顾名思义首先得安装 Java(JDK), 这边推荐 [Temurin](https://adoptium.net/zh-CN/temurin/releases/)

然后安装启动器(用来下载和管理游戏版本、模组等), 推荐 **[HMCL](https://github.com/HMCL-dev/HMCL)**(支持 m1)、[PrismLauncher](https://prismlauncher.org/)、**[FCL](https://github.com/FCL-Team/FoldCraftLauncher)**(适用于安卓移动端)、[PojavLauncher](https://pojavlauncherteam.github.io/)(适用于安卓、iOS)

### 基岩版(BE/PE)

- 1.20.51(当前推荐版本)

包括网易版和国际版, 推荐后者, 可在[苦力怕论坛](https://klpbbs.com/)下载.

> [!important]+ 建议修改存储为外部(否则在 Android/data 中无法查看存档)  
> 存档导出模组: https://llama-studios.github.io/mobile-export/

[Toolbox ζ 汉化版(支持1.20.51)版本更新！ - 软件资源 - Minecraft(我的世界)苦力怕论坛 (klpbbs.com)](https://klpbbs.com/thread-101678-1-1.html): 一个基岩版外挂, 可惜闪退用不了

### 光影

光影基于渲染模组, 主流的渲染模组为
- OptiFine(常用于 Forge)
- Iris(推荐, 常用于 Fabric)

> 推荐一个低配光影: MakeUp(支持 Iris, 比 BSL、[Photon](https://github.com/sixthsurge/photon)(直接下载 zip 文件即可) 帧率更好). 正常配置直接上 Complementary.

> 如遇模组效果不显示, 建议重载光影.

> 基岩版参考这个(建议放弃) https://mcpedl.com/newb-shader/

        - 一个充能的红石中继器激活在它前面的红石粉和机械，或者*强充能*一个红石导体。<ins>它对在它下面，上面，侧面，背面的方块无影响</ins>。

#### [充能与激活](https://zh.minecraft.wiki/w/%E7%BA%A2%E7%9F%B3%E7%94%B5%E8%B7%AF/%E5%85%85%E8%83%BD%E4%B8%8E%E6%BF%80%E6%B4%BB)

当一个方块被充能时，这个方块毗邻的所有机械元件都可以被激活，被充能的方块叫做充能方块。

能够被充能的方块就被称为红石导体（也叫做实体方块）。大多数红石导体都是不透明的方块。一些透明方块可布线(但不会被充能), 其中一些(半砖/台阶、漏斗、萤石)在垂直方向只能向上传输.
- **强充能**的方块可以激活毗邻的机械元件和红石元器件。红石导体可以被[电源](https://zh.minecraft.wiki/w/%E7%BA%A2%E7%9F%B3%E7%94%B5%E8%B7%AF#电源)及中继器、比较器强充能。<ins>被强充能的方块所表现性质和红石块大体相同</ins>，但是*红石块不是红石导体*，且**可以**改变红石线的形状和指向、向[红石比较器]( https://zh.minecraft.wiki/w/%E7%BA%A2%E7%9F%B3%E6%AF%94%E8%BE%83%E5%99%A8 "红石比较器")边侧输入信号[仅 Java 版]，而强充能方块不能。
- 当红石导体**仅被**红石线充能，我们称这个方块被**弱充能**。弱充能与强充能的区别在于，==弱充能方块无法激活毗邻的红石线==。

充能方块无法再去充能其他方块，只有电源和传输元件才可以去充能一个方块。

使用多少信号强度的信号去充能一个方块，我们就说这个方块有多少**充能等级**。被强充能的方块有多少充能等级，就可以激活毗邻的[红石粉]( https://zh.minecraft.wiki/w/%E7%BA%A2%E7%9F%B3%E7%B2%89 "红石粉")至多少信号强度。充能等级的大小与充能的强与弱完全无关。

红石导体在以下的任意一种情况中，都是被强充能了：
- 开启的电源的附着（如果有）。
- 输出信号的红石火把、红石比较器、红石中继器的指向。
- 开启的陷阱箱和正在翻页的讲台下方的方块。
- 发出红石信号的侦测器背朝的方块。
- 红石块本身（属于红石绝缘体）。

红石导体在以下的任意一种情况中，都是被弱充能了：
- 红石线的附着。
- 红石线的指向。

每个充能方块都可以传输信号到某些毗邻方块。但传递过程可能因方块本身不同而出现以下特定规则：
- 除测重压力板、陷阱箱之外的电源和红石中继器只能使红石导体强充能至15等级。
- 红石线是多少信号强度，被其充能的方块就被弱充能至多少充能等级。
- 测重压力板、陷阱箱和红石比较器输出的信号是多少强度，被充能的方块就被强充能至多少充能等级。

红石导体除了能被充能这一种性质外，还有使在激活的动力铁轨上的矿车加速离开另一端的红石导体[仅Java版]、允许红石比较器检测后方方块、“压断”红石线、阻止箱子开启的性质。红石线不能穿过红石导体与斜上方的红石线连接，箱子和末影箱不能在红石导体下方开启。


一个机械元件可以被其毗邻**激活**  
**激活该机械元件的充能方块的充能等级的大小并不会影响激活结果**

对于属于红石导体的机械元件（包括命令方块、投掷器、发射器、音符盒与红石灯等等），既可以被充能也可以被激活。因此区分它们是被激活还是被充能相当重要：

- 如果它能够激活毗邻的红石元器件和机械元件，那么它就被**充能**了。
- 如果它本身作出了一定的反应，那么它就被**激活**了。

机械元件只要被充能就一定也会被激活，但被激活并不会一定会被充能（如被毗邻的充能方块激活）。

不是红石导体的机械元件（门、栅栏门、活塞、漏斗、铁轨、活板门等等）可被激活并作出反应，但因为不具备红石导体的性质而无法被充能。

半连接激活: **发射器**、**投掷器**与**活塞**能够在三种机械当上方的方块_能被_激活时也会被激活，即使这个“上方的方块”并不存在也没问题。
