%%[list2table]%%
- < 层 Layer | 数据单元 | | 
- 物理层 Physical | 比特 Bit
    - 设备
        - 网卡, 网线, 调制解调器
        - 中继器、集线器
- 数据链路层 Data Link | 帧 Frame
    - 设备
        - 网桥/交换机
    - 协议
        - ARP, RARP
        - HDLC, PPP
- 网络层 Network | 包 Packet
    - 设备
        - 路由器
    - 协议
        - IP, ICMP, IGMP
        - RIP, OSPF, IGRP, BGP
- 传输层 Transport | 段 Segment
    - 设备
        - 网关
    - 协议
        - TCP
        - UDP
- 会话层 Session | 
    - 协议
        - SSL/TLS
        - DAP/LDAP
        - RPC
- 表示层 Presentation | 
- 应用层 Application | 报文 Message
    - 协议
        - HTTP, FTP, SMTP, POP3/IMAP, NNTP, TELNET/SSH, RTP
        - DNS, DHCP, NTP, SNMP
