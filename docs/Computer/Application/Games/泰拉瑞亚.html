困难模式前Boss（肉山前）​

Boss触发条件意义成就战前准备
史莱姆王- 自然：史莱姆雨击杀150只（首次）/75只（后续）
- 手动：史莱姆王冠（金冠+20凝胶）		🌟 首个Boss，提供史莱姆坐骑、忍者装备，无进度影响远程武器（三发猎枪）、赫尔墨斯靴、长平台风筝

- 手动：可疑眼球（6晶状体合成）		🌟 解锁魔矿/猩红矿，制作暗影/猩红套，推进武器升级铂金/金套（防御≥40可站撸）、篝火、再生药水、长平台
世界吞噬怪- 自然：腐化之地打破3个暗影珠
- 手动：蠕虫诱饵（腐肉+魔粉）		🌟 腐化世界专属，掉落暗影鳞片制作暗影套链球/蓝月（地牢）、阳炎之怒（地狱），针对多节体特性输出
克苏鲁之脑- 自然：猩红之地打破3个猩红之心
- 手动：血腥脊椎（脊椎+毒粉）		🌟 猩红世界专属，提供组织样本制作猩红套肌腱弓+小丑箭（一阶段）、迷你鲨（二阶段），保持距离
蜂王- 自然：破坏蜂巢幼虫
- 手动：憎恶之蜂（蜂蜜块+毒刺等）		🌟 掉落蜜蜂枪/养蜂人，大幅降低肉山难度猩红/暗影套+19防，配牛黄（毒免）、蜂蜜池回血，三发猎枪输出
骷髅王- 首次：夜晚与地牢老人对话选“诅咒”
- 后续：击杀穿巫毒娃娃的服装商		⛓️ 解锁地牢（否则地牢守卫秒杀），获取黄金箱装备（如魔法导弹）三层平台绕圈，肌腱弓/迷你鲨，防御堆至60+（配克盾防击退）
独眼巨鹿- 自然：雪地暴雪夜
- 手动：鹿华（小雪怪皮毛+晶状体）		🌟 《饥荒》联动Boss，掉落实用宠物（开猪猪存钱罐），无主线影响高防高回血装（甲虫壳/蠕虫围巾），跳起躲避打断攻击
血肉墙地狱中将向导巫毒娃娃丢入熔岩（向导需存活）⛓️ 开启困难模式核心：生成新三矿、神圣/腐化蔓延，解锁后期内容长矿车轨道（2000格），蜜蜂手雷/凤凰枪；或熔岩靴+熔岩套，铁皮/耐力药水

攻略​

作为泰拉瑞亚新手，你的核心目标应是 “安全度过前期 → 击败血肉墙开启困难模式”。以下是分阶段优先级目标和详细操作指南，结合旅途模式/小地图特性优化效率：

一、肉山前核心目标（5步递进）​

目标1：基础生存（第1-2天）​

  • 行动：
    1. 砍树收集200+木材（做工作台+木剑）。
    2. 建第一间房（6×10大小，桌椅+光源+门），吸引商人入住（卖基础武器）。
    3. 地表探索找宝箱（开武器如三发猎枪）和生命水晶（右键使用+20生命）。
  • 关键物品：木盔甲套装、钨/铅镐（挖第一级矿）。

目标2：装备升级（第3-4天）​

  • 行动：
    1. 挖矿做铁/铅套（防御≥20），优先升级镐（挖金/铂金矿）。
    2. 找沙漠做仙人掌剑（伤害15↑，前期神器）。
    3. 收集生命水晶堆血量至200+（解锁血月事件）。
  • 关键地点：地下浅层（0-300英尺），避开腐化/猩红。

目标3：首个Boss（第5-6天）​

  • 推荐Boss：史莱姆王（最弱） → 克苏鲁之眼（主线推进）。
  • 准备：
    • 铂金/金盔甲（防御≥25）。
    • 长平台（≥200格） + 再生药水 + 篝火。
    • 武器：三发猎枪（宝箱）或肌腱弓（猩红世界）。
  • 收益：魔矿/猩红矿 → 做暗影/猩红套（防御30↑）。

目标4：地牢与地狱（第7-10天）​

  • 行动：
    1. 击败骷髅王（夜晚对话老人）：
      • 准备：赫尔墨斯靴+克苏鲁护盾（防击退），金/铂金套。
    2. 探索地牢：拿黄金箱装备（钴盾、魔法导弹）。
    3. 下地狱挖狱石：
      • 准备：黑曜石皮药水（免疫熔岩）+ 熔岩靴。
      • 做熔岩套（防御35↑）和熔岩镐。

目标5：血肉墙（最终战）​

  • 准备：
    • 武器：蜜蜂手雷（蜂后掉落）或凤凰枪（地狱宝箱）。
    • 场地：地狱长直平台（≥400格，铺篝火/心灯）。
    • 配饰：熔岩靴+克苏鲁护盾+再生戒指。
  • 提示：向导存活时，将巫毒娃娃丢入熔岩召唤。

二、肉山后初期目标（困难模式生存）​

紧急任务：防蔓延+挖新矿​

  • 隔离带：在腐化/猩红与家园间挖6格宽通道（防蔓延）。
  • 新三矿：砸恶魔祭坛 → 世界生成钴/钯金 → 秘银/山铜 → 精金/钛金。
    • 优先做精金镐（挖叶绿矿）。

首轮Boss：机械三王（推荐顺序）​

  1. 毁灭者（多节体，最易）：

    • 武器：代达罗斯风暴弓（神圣宝箱怪）+ 圣箭（精灵尘+箭）。
    • 场地：高空平台（防地面触手），配护士回血（专家+）。
    • 收益：力量之魂 → 巨兽鲨（射手核心）。

  2. 双子魔眼：

    • 武器：巨兽鲨（毁灭者掉落）+ 水晶子弹。
    • 技巧：克苏鲁护盾冲刺躲激光，优先击杀激光眼。

  3. 机械骷髅王：

    • 武器：神圣剑（战士）或黄金雨（法师）。
    • 技巧：断手降低攻速，叶绿弹跟踪输出。
    • 意义：三王全败 → 解锁世纪之花！

三、新手必备技巧​

旅途模式专属优势​

  • 复制资源：研究过的矿石/木材无限取用，省去重复挖矿。
  • 敌怪难度0.5×：初期调低难度，装备成型后逐步调高。
  • 时间加速：24倍速跳过夜晚/事件等待时间。

通用生存法则​

  • 房屋集群：NPC房集中建，快速解锁森林晶塔（传送网核心）。
  • 备用逃生：随身带回忆药水（危险时回城）。
  • 战斗四件套：篝火+心灯+巴斯特雕像（+5防御）+ 蜂蜜池（回血加速）。

新手七日计划表（小地图适用）​

天数核心目标关键操作成果

第2天生命与防御挖铁矿做盔甲，找生命水晶（血量≥200）防御≥20，解锁血月事件

终极口诀​

“先活着，再变强，最后推进度”

  1. 木头火柴盒 → NPC入驻 → 晶塔网
  2. 挖矿做套 → 堆血200 → 虐史王
  3. 地牢拿宝 → 地狱熔岩套 → 肉山长跑道
  4. 隔离带保家 → 新三矿镐 → 机械三王按顺序打

按此流程，你将从“被僵尸追着跑”进阶到“硬刚月球领主”！ 🌟
