软件工具​

  • Karabiner: 开源改键软件, 支持鼠标
  • MacMouseFix: 支持滚动优化以及对除了左右键以外的键进行设置. 不能自定义手势, 不能针对 app 设置. 国区免费?
  • Mos: 一个开源的简单的鼠标滚动优化工具
  • BetterAndBetter: 大而全
  • xGestures: 一个老牌鼠标手势软件, 支持 AppleScript, 特点是支持修饰键控制(按需触发), 不足是不能同时支持右键、中键触发
  • MacGestures
  • MacStroke: 基于 MacGestures 的修改增强
  • WGestures2: 一个原 Windows 下的收费手势工具, 现支持了 mac, 功能上看着不错.

[!tip]+ 用 karabiner 把鼠标中键(button3)改为 shift, 把 xGestures 的修饰键设为 shift, 这样便能最大程度保留 MacMouseFix 功能的同时支持鼠标手势.
