AI

2024年11月6日 · 阅读需 8 分钟

概念​

%% 需包括所有重要概念, 按发展脉络顺序介绍, 例如 Prompt、RAG、Agent、MCP. %%

  • 机器学习(ML)：AI 的一个子集，使系统能够无需显式编程，而是使用算法和统计模型从数据中学习模式。
  • 深度学习(DL)：ML 的一个子领域，使用多层神经网络来建模图像、文本和语音等数据中的复杂模式。
  • Transformer 架构：2017 年引入的革命性神经网络架构。它使用一种称为"自注意力(Self-Attention)"的机制来理解序列数据中的上下文关系，构成了几乎所有现代大语言模型(LLM)的基础。
  • 大语言模型(LLM)：基于 Transformer 架构、在海量文本数据集上预训练的非常大(通常具有数十亿参数)的深度学习模型。它们表现出强大的通用语言理解和生成能力。
  • 提示工程(Prompt Engineering)：精心设计输入文本(提示)以有效引导 LLM 产生所需输出的实践。这是与基础模型交互的关键技能。
  • 检索增强生成(RAG)：一种通过从外部数据库或知识库检索相关信息来增强 LLM 知识的技术，然后再生成响应，从而提高准确性并减少幻觉。
  • AI 智能体(Agent)：一个使用 LLM 作为"大脑"的系统，可以感知环境、做出决策并通过规划和利用工具(如 API、数据库等)执行操作以自主实现给定目标。
  • 模型上下文协议(MCP)：由 Anthropic 开创的一种开放协议，旨在标准化 AI 模型(智能体)与外部工具、数据源和服务的交互方式。它通过提供通用标准，旨在解决 M 个客户端与 N 个服务之间的 O(M×N)集成复杂度问题，将其降低到 O(M+N)。

工具​

%% 需完整列举全生命周期生态工具链并归类, 例如 Ollama、n8n、Dify、Cherry Studio、Cursor 等 %%

模型部署​

  • Ollama: 一行命令拉取并本地运行 Llama、Qwen、Mistral 等 GGUF 模型, macOS/Linux/Win 支持.
  • LM Studio: 可视化桌面客户端, 可浏览、下载、聊天, 兼容 OpenAI API.
  • vLLM: 高吞吐、PagedAttention 生产级推理引擎, 支持 continuous batching.
  • SGLang: 2024 开源, RadixAttention + Chunked Prefill, 长上下文友好.
  • Xinference: 国产开源推理平台, 100+ 模型一键服务化, RESTful & gRPC.

应用编排 / LLMOps​

  • Hugging Face Transformers: Python 生态最丰富的预训练模型库.
  • LangChain: 链、记忆、工具调用、Agent 的通用框架.
  • LlamaIndex: 专注 RAG 的数据连接与索引框架.
  • CrewAI: 多智能体角色-任务-工具协作框架.
  • n8n: 开源低代码工作流, 节点拖拽即可连接 API、数据库、AI.
  • Dify: 开源 LLMOps 平台, Prompt IDE、RAG、Agent、一键发布 API.
  • Coze: 低代码聊天机器人平台, 与抖音、飞书生态深度集成.
  • Flowise: 拖拽式 LLM 流程构建器, 一键导出 Docker Compose.

监控与可观测​

  • LangSmith: LangChain 官方链路追踪、回归测试、在线标注.
  • Helicone: 开源 LLM 网关, 成本/延迟/Token 级别可视化, 支持 OpenAI & 本地 vLLM.

通用 Agent 平台 / IDE​

  • Cursor: VS Code 深度魔改, 集成 Claude / GPT 与 MCP 的 AI IDE.
  • GitHub Copilot: 主流编辑器内置 AI 代码补全与对话.
  • Windsurf: Codeium 出品, 多文件重构、命令行 Agent.
  • Continue: VS Code / JetBrains 开源 AI 插件.
  • Aider: CLI 多文件编辑 + 单元测试.
  • Cline: VS Code Agent 插件, 可执行命令.
