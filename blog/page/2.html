Emacs Vs Vim

2025年12月22日 · 阅读需 1 分钟

Vim 的特色在于多模态及其按键设计.

Emacs 是一整个由 Lisp 驱动的文本(buffer)处理系统, 而不只是一个编辑器.

就编辑器而言, Vim (的按键设定)更胜; 但论深度定制能力, Emacs 更强?

简而言之, Vim 是一个编辑器, Emacs 则是一整套文本处理系统.

Emacs 成也 Lisp, 败也 Lisp.
